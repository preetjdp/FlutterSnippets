# Flutter Screen Draw

This is a simple Demo that uses custom painter and Gesture Detector in Flutter to Doodle.

# Config

To Change the Color, size of the stroke just edit the main.dart file

```sh
Paint paint = Paint()
      ..color = Colors.black
      ..strokeCap = StrokeCap.round
      ..strokeWidth = 5.0;
```
